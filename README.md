# ⚓ Port Management API System

نظام تقني مصغر لإدارة أطقم الموانئ وجدولة الورديات (Shifts) باستخدام Node.js.

## 🛠️ تعليمات التشغيل (Setup)
لتشغيل المشروع على جهازك، نفذ السطر التالي في الترمينال:

1. تثبيت المكتبات، تجهيز الإعدادات، وتشغيل السيرفر:
   npm install && cp .env.example .env && npm run dev

2. المنفذ (Port): يعمل السيرفر افتراضياً على الميناء 4000.

---

## 🔗 أمثلة لروابط الوصول (API Endpoints)
يمكنك تجربة الروابط التالية لاختبار النظام:

- جلب بيانات الأطقم:
  GET http://localhost:4000/api/v1/crews

- جلب جداول الورديات:
  GET http://localhost:4000/api/v1/shifts

---

## 🆔 قاعدة توليد المعرفات (ID Generation)
تعتمد هذه النسخة من المشروع على قاعدة "الزيادة التلقائية" (Auto-Increment):
- عند إضافة أي عنصر جديد (وردية أو طاقم)، يقوم النظام بقراءة آخر معرف (ID) موجود وزيادته بمقدار رقم واحد (ID + 1).
- الهدف من هذه القاعدة هو ضمان وجود معرف رقمي فريد وسهل التتبع لكل سجل، مما يسهل عملية الربط بين الورديات والأطقم.

---

## 📂 ملاحظات تقنية
- تم استخدام مكتبة `dotenv` لإدارة متغيرات البيئة.
- تم تفعيل خاصية `watch` لمراقبة التغييرات في ملف .env لضمان تحديث البورت تلقائياً دون الحاجة لإعادة تشغيل السيرفر يدوياً.
